easyJet plc FY2025: Audit Risk Review

Thesis

A financial reporting and audit-planning review built around easyJet's published FY2025 statements. The project focuses on revenue cut-off, deferred revenue, lease-heavy capital structure, maintenance provisions, liquidity, and evidence-led analytical review, supported by an Excel workbook and Python-generated workings.

What this covers

This project was designed as recruiter-facing evidence of financial statement analysis, audit risk assessment, IFRS judgement awareness, and practical written communication. It is structured like an audit planning memo rather than a university essay.

Word report with audit-style commentary Professional write-up focused on audit risk, control considerations, and IFRS-aware observations.
PDF version for final presentation Formatted for recruiter review and easy sharing.
Excel workbook with raw data and ratios Underlying schedules, calculations, and analytical review workings.
Audit flags and support schedules Key risk areas, analytical anomalies, and evidence mapping.
Python scripts used to structure the workings Data processing and automated report generation code.
Direct file access for recruiter review All source documents and working papers accessible in one place.

The analysis is built around airline-specific realities rather than generic ratio templates, including customer prepayments, revenue recognition timing, lease liabilities, maintenance obligations, and liquidity sensitivity.
