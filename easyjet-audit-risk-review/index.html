Project Notes

What this demonstrates

This project was designed as recruiter-facing evidence of financial statement analysis, audit risk assessment, IFRS judgement awareness, and practical written communication. It is intentionally structured more like an audit planning memo than a university essay.

What is included

  • Word report with audit-style commentary
  • PDF version for final presentation
  • Excel workbook with raw data and ratios
  • Audit flags and support schedules
  • Python scripts used to structure the workings
  • Direct file access for recruiter review

The analysis is built around airline-specific realities rather than generic ratio templates, including customer prepayments, revenue recognition timing, lease liabilities, maintenance obligations, and liquidity sensitivity.
