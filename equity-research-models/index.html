Tesco PLC (TSCO LN): Integrated Model & Equity Research Note

Tesco is the UK's largest grocer with roughly 27% market share. The 12-tab integrated model covers revenue (UK & ROI + Central Europe), EBITDA, full PP&E and ROU asset rollforwards, IFRS 16 lease liability schedules, a debt rollforward, dividend and buyback modelling, a DCF with mid-year convention and Gordon Growth terminal value, and a 6-peer trading comps table. All inputs are sourced from Tesco Annual Reports FY2020-FY2024.

Model Architecture

12-tab structure (both models)

  • Cover page with company overview
  • Assumptions tab (blue inputs, black formulas, green links)
  • Income Statement (segmental revenue build)
  • Working Capital schedule (DIO / DSO / DPO)
  • Capex & D&A (PP&E and ROU rollforwards)
  • Debt & Interest (bond + IFRS 16 lease rollforward)
  • Dividend & Buyback (share count roll)
  • Balance Sheet with audit check
  • Cash Flow Statement
  • DCF Valuation (WACC/TGR sensitivity table)
  • Trading Comps (sector peers)
  • Charts tab

Key features

  • Every cell contains a live Excel formula, not hardcoded values
  • Base / Bull / Bear scenario toggle via CHOOSE() function
  • WACC from CAPM (10yr gilt, Bloomberg beta, Damodaran ERP)
  • Mid-year discounting convention
  • Source notes citing specific pages from annual reports
  • Python validation scripts verify all historical inputs

Technical stack

Excel models built with Python (openpyxl). Word documents generated with Node.js (docx-js). Data validation scripts use Python (pandas, numpy). All code is available in the GitHub repository.
