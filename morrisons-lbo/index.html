A fully formula-driven LBO model and professional transaction case study analysing CD&R's take-private acquisition of Wm Morrison Supermarkets for 287p per share, implying an enterprise value of approximately £9.9 billion. The project covers the full investment thesis, value creation plan, risk assessment, and returns analysis across three scenarios.

Model structure

7 tabs, 422 formulas, zero errors

  • Assumptions: deal overview, entry valuation, LTM financials, operating and exit assumptions
  • Sources & Uses: financing structure with leverage metrics
  • Operating Model: 5-year revenue, EBITDA margin ramp (6.1% to 7.2%), EBIT, tax, UFCF
  • Debt Schedule: 4 tranches (TLB, Secured Notes, Unsecured Notes, PIK Pref Shares), mandatory amort, cash sweep at 50%, credit metrics (ICR, FCCR, DSCR)
  • Returns: downside/base/upside scenarios, MoM, IRR via =IRR(), reconciling value creation bridge
  • Sensitivity: two-way tables for IRR and MoM
  • Summary: one-page dashboard

Transaction note

Professional case study covering the full investment thesis and transaction analysis:

  • Executive summary — Overview and key investment highlights
  • Investment thesis — Vertical integration, property portfolio, margin recovery opportunity, and comparable transactions
  • Industry context — UK grocery market structure, competitive dynamics, and consumer trends
  • Value creation plan — Strategic initiatives with supporting bridge table showing source and application of value
  • Key risks — Downside scenarios and stress factors to profitability and exit multiples
  • Returns summary — Three-scenario analysis with detailed scenario table (downside, base, upside)
  • Conclusion — Investment recommendation and forward-looking assessment
