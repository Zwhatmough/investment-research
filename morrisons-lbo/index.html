LBO Model & Transaction Case Study

LBO PROJECT 01 April 2026
Consumer / Retail / Private Equity

CD&R Take-Private of Wm Morrison Supermarkets plc (2021)

A fully formula-driven LBO model and professional transaction note analysing CD&R's acquisition of Morrisons for 287p per share, implying an enterprise value of approximately £9.9 billion. The project covers the investment thesis, value creation plan, risk assessment, and returns analysis across downside, base, and upside scenarios.

Model structure

7 tabs, 422 formulas, zero errors. Assumptions, Sources & Uses, Operating Model (5-year projections with margin ramp from 6.1% to 7.2%), Debt Schedule (4 tranches: TLB, Secured Notes, Unsecured Notes, PIK Pref Shares with cash sweep mechanics), Returns (IRR via =IRR(), reconciling value bridge), Sensitivity (two-way tables), and Summary dashboard.

Transaction note sections

Executive summary, investment thesis (vertical integration, property portfolio, margin recovery, comparable transactions including Asda/TDR Capital and Tesco-Booker), industry context, value creation plan with bridge table, risk analysis, returns summary with scenario table, and a conclusion with investment recommendation.
